# G-Central - Your Central Station for Fun and Games
Built with pure HTML, css and JavaScript.

## What is G-Central?
- G-Central is a hub of basic but enjoyable games that can be played on your browser.
- It features a growing catalogue of classic and new games.
- A featured game, usually a new addition or a fan favorite, is also displayed at the top of the screen.

## Getting Started
### Github Deployment
Since this project is built purely with HTML, css and JavaScript, github automatically hosts it here: https://waseempt.github.io/gcentral

### Local Development
If you wish to run this website and develop it locally, here are the steps to doing that:
  1) Clone the repository to your own machine,open a terminal in your desired location and use this command: 
     `git clone https://github.com/waseempt/gcentral.git`.
  2) If you simply want to run the website locally, all you have to do is open the 'index.html' file in your browser.
  3) For a better development experience, I suggest opening the project with your favorite code editor/IDE, and using a live server extension (e.g.: VS code with the 'Live Server' Extension).
